# Using Data to Improve MLB Attendance

## Overview

Major League Baseball organizations continuously seek ways to increase attendance, improve fan engagement, and maximize revenue. This project analyzes attendance patterns, game characteristics, and event-related factors to identify opportunities for increasing attendance.

The analysis transforms historical attendance data into actionable business recommendations.

## Business Problem

Declining or inconsistent attendance can negatively impact:

- Ticket revenue
- Concession sales
- Merchandise purchases
- Fan engagement

Organizations require data-driven insights to understand which factors most strongly influence attendance outcomes.

## Project Objectives

- Analyze attendance trends
- Identify attendance drivers
- Evaluate event-related impacts
- Generate actionable recommendations
- Support fan engagement strategies

## Results

The analysis identified several factors associated with increased attendance and generated business-focused recommendations to improve fan turnout and engagement.
